DevUNO

A Multipurpose Arduino Compatible IoT Industrial Development Board

Description

DevUNO is a multipurpose Arduino development board that allows you to quickly and easily test all your existing modules, sensors, and components without having to struggle with breadboards and "cobwebs". It can be immediately deployed in real-time/world.

DevUNO covers a wide area of use, namely: industry, agriculture, animal husbandry, scientific research, and educational projects. It is also a great gadget for hobbyists.

Technical Specifications

  • ATmega328P 8-bit AVR Microcontroller
  • 32 KB Flash Memory, 2 KB SRAM, 1 KB EEPROM
  • 16 MHz Clock Speed
  • 14 Digital I/O Pins (6 PWM outputs)
  • 6 Analog Inputs
  • Extended Digital Ports: D3, D5, D6, D7, D8, D9 (6 ports)
  • Extended Analog Ports: A5, A4, A3, A2, A1, A0 (6 ports)
  • Extended I2C Ports: 6x ports via A4 and A5 (6 ports)
  • Integrated 5V/2A DC Power Supply
  • Integrated Voltage Divider on A2/VIN and A3/VOT for Power Supply Measurements
  • Integrated RF 433MHz OOK/ASK Wireless Module - Receiver / Transmitter (optional)
  • Standard 2.54mm / JST-XH Headers with access to I/Os, SPI, UART, ADC, ISP (ICSP), and Power pins
  • Arduino UNO Shield Compatible
  • Input voltage: 9-24V DC (max. 30V DC)
  • Operating voltage: 5V DC (has an integrated 3.3V / 800mA LDO voltage regulator)
  • Dimensions: 117 mm x 96 mm
  • Weight: 121 g (with Transparent Plexiglas base)
  • Operating temperature: -40 – 85 °C

Extended Specifications

It offers you the possibility to choose the Clock source of the Microcontroller, i.e., to use a Crystal, a Resonator, or the Internal Oscillator of the Microcontroller itself (in that case, all you need to do is to disable the External Clock source by shorting out the jumper pads on the board’s backside).

Microcontroller Compatibility

Can be used with any of the following AVR Microcontrollers:

  • ATtiny48
  • ATtiny88
  • ATmega8 (Arduino NG – 8s delay by Bootloader)
  • ATmega48
  • ATmega88
  • ATmega168
  • ATmega328 (Arduino UNO)

Setup Instruction

To setup the Microcontroller follow the steps:

  1. Download and install Arduino IDE for Programming, Debugging, Compiling, and Uploading.
  2. Connect your favorite USB 2.0 to TTL UART 5-pin Adapter (CP2102, CP2104, FT232RL) to connect DevUNO’s UART port with your computer using Arduino IDE.
  3. Start Arduino IDE, open or write your desired code.
  • Click Tools -> Board -> and select Arduino/Genuino UNO.
  • Click Tools -> Port -> and select the COM port as detected by the computer in the device manager.
  • Click Verify to compile the code.
  • Click Upload to burn the code / ".hex" file into microcontroller.
